API Endpoints

Once deployed, the API will be available at /v1/agents with the following endpoints:

  • GET /v1/agents - Search agents with filters
  • GET /v1/agents/{id} - Get agent details
  • POST /v1/ai/select - AI-powered recommendations
  • POST /v1/search - Advanced search with JSON body
